# Cinematek Movie Search App

## Overview
Cinematek is a web-based movie search application that allows users to explore new and top-rated movies, search for specific titles, and manage a favorites list. Built with HTML, JavaScript, and Tailwind CSS, it features a responsive design, multilingual support (English, Persian, Chinese), and a dark/light theme toggle. The app fetches data from an external API and includes a hero carousel for showcasing top movies.

## Features
- **Multilingual Support**: Interface available in English, Persian, and Chinese with dynamic language switching.
- **Search Functionality**: Search movies by title with real-time results.
- **Filters and Sorting**: Browse new movies, top-rated movies, or favorites, with sorting options by rating or year.
- **Favorites Management**: Add or remove movies from a favorites list, stored locally.
- **Responsive Design**: Optimized for both mobile and desktop devices.
- **Theme Toggle**: Switch between dark and light themes.
- **Hero Carousel**: Displays top-rated movies with details and navigation.
- **Interactive UI**: Includes modals for movie details, star ratings, and smooth animations.

## Installation
1. **Download the Project**: Download the project files from the repository.
2. **Host Locally**: Use a local server (e.g., Live Server in VS Code or `http-server`) to serve the `index.html` file.
3. **Dependencies**: No additional installations are required as the app uses CDN-hosted Tailwind CSS and fonts (Vazirmatn, Noto Sans SC, Roboto).
4. **API Access**: Ensure an active internet connection to fetch movie data from the external API (`https://cinemaplus-app.vercel.app`).

## Usage
1. Open `index.html` in a web browser via a local server.
2. Use the search bar to find movies by title.
3. Browse movies using the filter buttons (New Movies, Top Rated, Favorites).
4. Sort results by rating or release year using the dropdown menu.
5. Click a movie card to view details in a modal, including plot and download links (if available).
6. Add/remove movies from favorites by clicking the heart icon.
7. Switch languages using the dropdown in the navigation bar.
8. Toggle between dark and light themes using the theme button.

# اپلیکیشن جستجوی فیلم سینماتِک

## مرور کلی
سینماتِک یک اپلیکیشن وب برای جستجوی فیلم است که به کاربران امکان می‌دهد فیلم‌های جدید و برتر را کاوش کنند، عناوین خاصی را جستجو کنند و فهرست علاقه‌مندی‌ها را مدیریت کنند. این اپلیکیشن با استفاده از HTML، جاوااسکریپت و Tailwind CSS ساخته شده و دارای طراحی پاسخ‌گو، پشتیبانی از چند زبان (انگلیسی، فارسی، چینی) و قابلیت تغییر تم روشن/تیره است. داده‌ها از یک API خارجی دریافت می‌شوند و شامل یک کاروسل ویژه برای نمایش فیلم‌های برتر است.

## ویژگی‌ها
- **پشتیبانی چندزبانه**: رابط کاربری در زبان‌های انگلیسی، فارسی و چینی با امکان تغییر زبان پویا.
- **قابلیت جستجو**: جستجوی فیلم‌ها بر اساس عنوان با نتایج آنی.
- **فیلتر و مرتب‌سازی**: مرور فیلم‌های جدید، برتر یا علاقه‌مندی‌ها با گزینه‌های مرتب‌سازی بر اساس امتیاز یا سال انتشار.
- **مدیریت علاقه‌مندی‌ها**: افزودن یا حذف فیلم‌ها از فهرست علاقه‌مندی‌ها که به‌صورت محلی ذخیره می‌شود.
- **طراحی پاسخ‌گو**: بهینه‌شده برای دستگاه‌های موبایل و دسکتاپ.
- **تغییر تم**: جابه‌جایی بین تم‌های روشن و تیره.
- **کاروسل ویژه**: نمایش فیلم‌های برتر با جزئیات و ناوبری.
- **رابط کاربری تعاملی**: شامل مودال برای جزئیات فیلم، امتیاز ستاره‌ای و انیمیشن‌های روان.

## نصب
1. **دانلود پروژه**: فایل‌های پروژه را از مخزن دانلود کنید.
2. **میزبانی محلی**: از یک سرور محلی (مانند Live Server در VS Code یا `http-server`) برای اجرای فایل `index.html` استفاده کنید.
3. **وابستگی‌ها**: نیازی به نصب اضافی نیست، زیرا اپلیکیشن از Tailwind CSS و فونت‌ها (وزیرمتن، Noto Sans SC، Roboto) از طریق CDN استفاده می‌کند.
4. **دسترسی به API**: اطمینان حاصل کنید که اتصال اینترنت فعال است تا داده‌های فیلم از API خارجی (`https://cinemaplus-app.vercel.app`) دریافت شود.

## استفاده

3. با استفاده از دکمه‌های فیلتر (فیلم‌های جدید، فیلم‌های برتر، علاقه‌مندی‌ها) فیلم‌ها را مرور کنید.
4. نتایج را بر اساس امتیاز یا سال انتشار از طریق منوی کشویی مرتب کنید.
5. روی کارت فیلم کلیک کنید تا جزئیات آن در مودال، شامل خلاصه داستان و لینک‌های دانلود (در صورت وجود) نمایش داده شود.

7. با استفاده از منوی کشویی در نوار ناوبری، زبان را تغییر دهید.

# 电影天堂电影搜索应用

## 概述
电影天堂是一个基于网络的电影搜索应用，允许用户探索新片和评分最高的电影，搜索特定标题，并管理收藏列表。该应用使用HTML、JavaScript和Tailwind CSS构建，具有响应式设计，支持多语言（英语、波斯语、汉语），并提供暗/亮主题切换功能。应用从外部API获取数据，并包含一个用于展示顶级电影的英雄轮播。

## 功能
- **多语言支持**：界面支持英语、波斯语和汉语，可动态切换语言。
- **搜索功能**：通过标题搜索电影，实时显示结果。
- **过滤和排序**：浏览新电影、高评分电影或收藏夹，支持按评分或年份排序。
- **收藏管理**：将电影添加或移除出收藏列表，数据本地存储。
- **响应式设计**：优化适配移动设备和桌面设备。
- **主题切换**：在暗色和亮色主题之间切换。
- **英雄轮播**：展示高评分电影，包含详细信息和导航。
- **交互式界面**：包括电影详情弹窗、星级评分和流畅动画。

## 安装
1. **下载项目**：从仓库下载项目文件。
2. **本地托管**：使用本地服务器（例如VS Code中的Live Server或`http-server`）运行`index.html`文件。
3. **依赖项**：无需额外安装，应用通过CDN使用Tailwind CSS和字体（Vazirmatn、Noto Sans SC、Roboto）。
4. **API访问**：确保网络连接正常，以便从外部API（`https://cinemaplus-app.vercel.app`）获取电影数据。

## 使用
1. 通过本地服务器在浏览器中打开`index.html`。
2. 使用搜索栏按标题查找电影。
3. 使用过滤按钮（新电影、高评分、收藏夹）浏览电影。
4. 通过下拉菜单按评分或发行年份排序结果。
5. 点击电影卡查看弹窗中的详细信息，包括剧情和下载链接（如果有）。
6. 点击心形图标添加或移除电影到收藏夹。
7. 使用导航栏中的下拉菜单切换语言。
8. 使用主题按钮在暗色和 روشن主题之间切换。
